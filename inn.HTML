The long-awaited Worldcoin (WDC) 2.0.0, code name "Dancing Hippo", has been released. Worldcoin is a proof-of-work (scrypt) cryptocurrency forked from Bitcoin in May 2013. This major new release of the cryptocurrency's wallet client is called Worldcoin Business Centre (WBC or WorldcoinBC) and it features a complete overhaul of the software application's user interface as well as a host of new features: "Refactored architecture - now there are 4 completely separated layers. The Daemon - this is the application that performs crypto related activities like signing, sending coins, etc, in other words, an extremely stable stripped-down wallet. The Director - this is another application written in C++, its main objective is to translate, coordinate and send back an answer of requests from the UI to the appropriate component target. Components - all components are scripts. User interface - all elements of the UI are made using scripts and are loaded by the Director." Read the release announcement and release notes (PDF) for more information and upgrade instructions.
